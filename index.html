主要機能

💪 ワークアウト記録

  • 種目・重量・回数・セット数をすばやく記録
  • 前回値を見ながら、その場で次のセットを判断しやすい
  • インターバルタイマー内蔵
  • 63種目のプリセット＋カスタム種目に対応
  • Proではプレート計算機を利用可能
  • Quickly log exercise, weight, reps, and sets
  • See previous values so the next set is easier to decide
  • Built-in rest timer
  • 63 preset exercises + custom exercises
  • Plate calculator available on Pro

📊 進捗・履歴分析

  • 週・月・3か月・全期間で履歴を集計
  • 種目別の最大重量・推定1RM・総ボリュームで伸び方を見返せる
  • 目標の進捗確認と停滞アラート
  • Proでは全履歴・1RMランク・CSVエクスポートを利用可能
  • Aggregate history by week, month, 3 months, or all time
  • Review growth with max weight, estimated 1RM, and total volume by exercise
  • Goal progress tracking and stagnation alerts
  • Pro adds full history, 1RM rank, and CSV export

⚖️ 体組成トラッキング

  • 体重・体脂肪率・筋肉率・ウエストを記録
  • BMI・除脂肪体重・脂肪量を自動計算
  • 体重・体脂肪率はApple Healthへ保存時に書き出し、必要に応じて取り込み可能
  • Track weight, body fat %, muscle %, and waist
  • Auto-calculate BMI, lean mass, and fat mass
  • Write weight and body fat to Apple Health on save, and import the latest values when needed

👥 ソーシャル

  • Trero ID・QRコードでフレンド追加
  • フレンドのワークアウトを閲覧し、リアクションで継続を後押し
  • 週次ストリークで今週の達成状況を確認
  • Add friends via Trero ID or QR code
  • View friends' workouts and react to support consistency
  • Check weekly streak and current-week progress

Apple Watch対応。手首からセットを記録し、iPhone側の履歴へ反映できます。
