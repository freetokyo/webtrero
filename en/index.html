Trero is a strength training app that helps you log today’s workout while checking the weight and reps from your last session at the gym. Goal setting, progress analysis, body metrics, Apple Watch logging, and friend features are all in one place, so it is easier to stay consistent without adding more logging work.

What You Can Do

The main features people often check before installing are organised by the situations where they are actually useful. Instead of only listing features, this section explains which actions reduce which kinds of friction.

💪 When You Want Fast Logging

  • You can log exercises, weight, reps, and sets on iPhone.
  • When you select an exercise, the previous log appears, so you do not need to search separately for what you lifted last time.
  • If you use "same as last time," previous values can be filled in with one tap.
  • The rest timer can be used together with set saving and also supports notifications.

⭐ When You Reuse the Same Exercises

  • The app shows the top 5 most-used exercises from the last 7 days.
  • If you repeat the same routine often, it becomes easier to start without searching the full list each time.
  • In addition to 63 preset exercises, you can also create custom exercises.

🎯 When You Want Goal-Oriented Progress

  • You can set target weight, target reps, increase and decrease steps, and a deadline for each exercise.
  • You can check your current position and the next suggestion while reviewing progress rate.
  • When progress stalls, the app shows an adjustment idea such as changing weight or reps.
  • Your weekly goal can be changed from 1 to 7 workout days per week.

📊 When You Want to Review Progress

  • You can review history by week, month, 3 months, or all time.
  • You can check max weight, total volume, and estimated 1RM by exercise.
  • Weekly streaks and the calendar help you understand whether you are staying consistent.
  • Pro also includes CSV export, the plate calculator, and estimated 1RM rank.

⚖️ When You Track Body Changes Too

  • You can record weight, body fat percentage, muscle percentage, waist, and notes.
  • BMI, lean body mass, and fat mass are calculated automatically.
  • Weight and body fat percentage can be written to Apple Health when you save.
  • Import from Apple Health can be triggered manually from the body-record screen.

⌚ When You Want Better Consistency

  • You can log sets from Apple Watch and reflect them in iPhone history.
  • Friends can be added by Trero ID search or QR code scan.
  • You can send reactions to workouts and create a reason to keep logging.
  • Block and report are also supported.

Usage Examples

These examples show the kinds of users Trero fits well and how they can use it in practice.

Strength Goal

For Someone Chasing a 100 kg Bench

  • First, set a target weight and target reps for bench press.
  • For the first session, select bench press and start logging.
  • From the next session onward, you can quickly choose the same exercise while checking recent exercises and previous values.
  • While reviewing goal progress and the next suggestion, you can increase the load little by little.
Fixed Routine

For Someone Repeating the Same 5 Exercises

  • Your most-used exercises from the last 7 days appear near the top.
  • By choosing from that list each time, you can reduce the number of steps before logging starts.
  • If the set is the same as last time, the main flow becomes reusing previous values and saving.
  • If you also use the rest timer, logging and rest management can be handled in the same screen flow.
Body Management

For Someone Tracking Training and Body Composition

  • You can review workout history together with changes in weight and body fat percentage in the same app.
  • Body composition can be recorded every day or only when needed.
  • If you use Apple Health, you can combine write-on-save with manual import.
  • This fits people who do not want to separate strength progress from body changes.
Consistency Support

For Someone Who Wants Social Accountability

  • Once you add friends, you can view their workouts.
  • By sending reactions, you can create a reason to keep recording.
  • Weekly streaks and weekly summary notifications also help you understand your own consistency.
  • If needed, block and report tools are available.

Common Questions

Can I use it as a guest? Yes. You can still log workouts as a guest. If you want to use another device or plan to keep using the app long term, moving later to an email or Apple account is safer.
How does Apple Health integration work? Weight and body fat percentage can be written to Apple Health when you save. Import is done manually from the body-record screen.
What can Apple Watch do? You can log sets from Apple Watch and reflect them in iPhone history.
What changes on the free plan? The free plan includes limits on history range, custom exercise count, and active goals. CSV export, the plate calculator, and estimated 1RM rank are Pro features.
