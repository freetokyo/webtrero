Trero is a strength training app that helps you log today’s workout while checking the weight and reps from your last session at the gym. Goal setting, progress analysis, body metrics, Apple Watch logging, and friend features are all in one place, so it is easier to stay consistent without adding more logging work.

Vad du kan göra

De viktigaste funktionerna som folk kollar innan installation är organiserade efter de situationer där de faktiskt är användbara. Istället för att bara lista funktioner förklarar detta avsnitt vilka åtgärder som minskar vilka typer av friktion.

💪 När du vill logga snabbt

  • Du kan logga övningar, vikt, reps och set på iPhone.
  • När du väljer en övning visas föregående logg – ingen separat sökning behövs.
  • Med "Samma som förra gången" kan tidigare värden fyllas i med ett tryck.
  • Viletimern kan användas tillsammans med setsparande och stöder notiser.

⭐ När du upprepar samma övningar

  • Appen visar de 5 mest använda övningarna från de senaste 7 dagarna.
  • Ju oftare du upprepar samma rutin, desto snabbare kan du komma igång utan sökning.
  • Utöver 63 förinställda övningar kan anpassade övningar skapas.

🎯 När du vill ha målstyrd framgång

  • För varje övning kan du ange målvikt, målreps, stegstorleken för ökningar och minskningar samt en valfri deadline.
  • Nuvarande position och nästa rekommendation är synliga vid granskning av framstegsprocenten.
  • Vid stagnering visar appen justeringsförslag som lägre vikt eller fler reps.
  • Det veckliga målet kan ställas in på 1–7 träningsdagar/vecka.

📊 När du vill granska framsteg

  • Granska historik per vecka, månad, 3 månader eller totalt.
  • Hämta maxvikt, totalt volym och beräknad 1RM per övning.
  • Veckliga streaks och kalender visar om du håller dig konsekvent.
  • Pro lägger till CSV-export, skivkalkylator och beräknad 1RM-rankning.

⚖️ När du även spårar kroppsförändringar

  • Registrera vikt, fettprocent, muskelprocent, midja och anteckningar.
  • BMI, fettfri massa och fettmassa beräknas automatiskt.
  • Vikt och fettprocent kan skrivas till Apple Health vid sparande.
  • Import från Apple Health kan startas manuellt från kropp-skärmen.

⌚ När du vill ha bättre konsekvens

  • Logga set från Apple Watch och återspegla dem i iPhone-historik.
  • Lägg till vänner via Trero-ID-sökning eller QR-kod.
  • Reaktioner på träning skapar en anledning att fortsätta.
  • Blockering och rapportering stöds också.

Användningsexempel

Dessa exempel visar vilka typer av användare Trero passar bra för och hur de kan använda det i praktiken.

Styrke mål

För den som jagar 100 kg bänkpress

  • Ange först målvikt och reps för bänkpress.
  • Välj bänkpress i första sessionen och börja logga.
  • Från nästa session väljer du snabbt samma övning från senaste övningar och kollar tidigare värden.
  • Öka gradvis belastningen vid granskning av målfremstegen.
Fast rutin

För den som upprepar samma 5 övningar

  • De mest använda övningarna från de senaste 7 dagarna visas överst.
  • Att välja från den här listan minskar antalet steg innan loggning.
  • Är setet detsamma som förra gången består huvudflödet av fyll i tidigare värden och spara.
  • Med viletimern kan loggning och vilohantering ske i samma skärmflöde.
Kroppsstyrning

För den som spårar träning och kroppssammansättning

  • Granska träningshistorik tillsammans med vikt- och fettförändringar i en app.
  • Registrera kroppssammansättning dagligen eller efter behov.
  • Kombinera skriv-vid-sparande och manuell import med Apple Health.
  • Idealiskt för dem som inte vill separera styrkefremsteg och kroppsförändringar.
Konsekvenstöd

För den som vill ha socialt ansvar

  • Granska vänners träning efter tillläggning.
  • Reaktioner skapar en anledning att fortsätta logga.
  • Veckliga streaks och veckosammanfattnings-notiser hjälper dig förstå din konsekvens.
  • Blockeringsand rapporteringsfunktioner finns tillgängliga vid behov.

Vanliga frågor

Hur fungerar Apple Health-integration? Vikt och fettprocent kan skrivas till Apple Health vid sparande. Import görs manuellt från kropp-skärmen.
Vad kan Apple Watch göra? Du kan logga set från Apple Watch och återspegla dem i iPhone-historik.
Vad ändras på Free-planen? Free-planen har begränsningar för historikräckvidd, anpassade övningar och aktiva mål. CSV-export, skivkalkylator och beräknad 1RM-rankning är Pro-funktioner.
